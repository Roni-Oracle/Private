## Per-Question QA

| קריטריון | שאלת בדיקה |
|---|---|
| יעד למידה | האם ברור מה השאלה בודקת? |
| רלוונטיות | האם השאלה מבוססת על חומר הלימוד? |
| בהירות | האם גוף השאלה ברור לפני קריאת התשובות? |
| חד-משמעיות | האם יש תשובה אחת מיטבית בלבד? |
| הוגנות | האם סטודנט שלמד היטב יכול לענות עליה? |
| רמת חשיבה | האם השאלה בודקת את רמת החשיבה שתוכננה? |
| ניסוח | האם אין עמימות, סרבול או שלילה מיותרת? |
| מסיחים | האם כל המסיחים סבירים? |
| אורך אפשרויות | האם ארבע האפשרויות דומות באורך, פירוט, סגנון ומבנה דקדוקי? |
| רמזים | האם אין רמז טכני לתשובה הנכונה? |
| עצמאות | האם השאלה אינה תלויה בשאלה אחרת? |

## Distractor QA

For every question, verify:

- Each distractor represents a plausible misunderstanding.
- No distractor is absurd, off-topic, or stylistically unlike the correct answer.
- No two distractors repeat the same idea.
- No distractor is partially correct enough to create a second plausible best answer.
- No distractor can be eliminated by length, grammar, punctuation, specificity, or wording alone.
- The correct answer is not consistently the longest or most detailed option.

## Whole-Exam QA

| היבט | שאלת בקרה |
|---|---|
| כיסוי החומר | האם כל נושאי הקורס המרכזיים מיוצגים? |
| משקל יחסי | האם הנושאים מופיעים לפי חשיבותם בקורס? |
| רמות חשיבה | האם יש איזון בין ידע, הבנה, יישום וניתוח? |
| רמת קושי | האם יש איזון בין קל, בינוני וקשה? |
| עומס קריאה | האם יש יותר מדי שאלות ארוכות? |
| כפילויות | האם יש שאלות שבודקות אותו דבר כמעט באותו אופן? |
| רמזים בין שאלות | האם שאלה אחת חושפת תשובה לשאלה אחרת? |
| פיזור תשובות | האם א, ב, ג, ד מפוזרות באופן מאוזן? |
| אחידות | האם המונחים והניסוחים עקביים לאורך המבחן? |
| הוגנות | האם המבחן בודק הבנה ולא מלכודות? |

## Fix Actions

| סוג הבעיה | פעולה נדרשת |
|---|---|
| ניסוח עמום | נסח מחדש את גוף השאלה |
| שתי תשובות אפשריות | חדד את ההקשר או החלף אפשרות |
| מסיח חלש | החלף במסיח סביר יותר |
| תשובה נכונה ארוכה מדי | קצר אותה או הרחב את המסיחים במידע מקביל |
| שאלה קלה מדי | הפוך אותה לשאלת הבנה, יישום או הבחנה |
| שאלה קשה בגלל ניסוח | פשט את הניסוח בלי להוריד את רמת החשיבה |
| שאלה על פרט שולי | החלף בשאלה על עיקרון מרכזי |
| שאלה כפולה | פצל, שנה או מחק |
| שאלה שחוזרת על שאלה אחרת | שנה יעד למידה או החלף |
| רמז טכני | איזן אורך, סגנון ודקדוק |
| תלות בשאלה אחרת | הפוך את השאלה לעצמאית |

## Hebrew And RTL QA

| קריטריון | תקין כאשר |
|---|---|
| כיוון כתיבה | כל הפסקאות, השאלות והאפשרויות מוצגות מימין לשמאל |
| יישור | הטקסט, הכותרות והתשובות מיושרים לימין |
| אותיות תשובה | כל האפשרויות מסומנות באותיות א, ב, ג, ד |
| פסיקים | הפסיק צמוד למילה שלפניו ואחריו רווח אחד |
| נקודות | הנקודה צמודה למילה שלפניה |
| סימני שאלה | סימן השאלה צמוד למילה האחרונה בשאלה |
| אנגלית בתוך עברית | מונחים באנגלית משולבים ללא רווח שגוי לפני הפיסוק |
| תבליטים | התבליטים והמספור מופיעים בצד ימין כאשר הפורמט מאפשר זאת |
| רווחים | אין רווחים כפולים או רווחים לפני סימני פיסוק |
| טבלאות | כותרות בעברית וסדר עמודות נוח לקורא עברית |
| אחידות | אותם מושגים נכתבים באותו אופן בכל המבחנים |

## Final QA Report Fields

For each exam, report:

- מספר שאלות.
- מספר שאלות ידע, הבנה, יישום, וניתוח / הערכה.
- מספר שאלות קלות, בינוניות, וקשות.
- מספר שאלות שתוקנו או הוחלפו.
- בעיות מרכזיות שאותרו.
- פעולות תיקון שבוצעו.
- הערכה כוללת לאחר תיקון.
